# Tech Stack

## Frontend Technologies
- **Vanilla ES6+ JavaScript**: Core application logic.
- **HTML5 Canvas API**: Rendering engine for both 3D perspective and 2D UI elements.
- **CSS3**: Layout, styling, and CRT/hit-flash animations.

## Build & Tooling
- **Vite**: Fast development server and production bundler.
- **Node.js**: Environment for running asset generation scripts.
- **pngjs**: Used in scripts to manipulate/generate PNG assets (`generate_classic_assets.js`, etc.).
