# Retro Adventure (MONOLITH)

## Overview

Monolith is a professional, high-performance pseudo-3D dungeon crawler engine built with Vanilla JavaScript and HTML5 Canvas. It features a "1990s Modern" textured aesthetic, combining classic DRPG mechanics with a custom rendering pipeline.

## Goals

1. Provide a smooth, browser-based pseudo-3D experience.
2. Maintain a 1990s retro aesthetic using raster textures and pixelated graphics.
3. Build a flexible foundation for a classic grid-based dungeon crawler (DRPG).

## Core User Flow

1. Start the game from the main menu.
2. Explore a grid-based dungeon in a first-person perspective.
3. Fight monsters, collect items, and interact with the environment (levers, doors).
4. Descend through floors and progress the character.

## Features

### Graphics Engine
- Raw Retro Vector Engine with geometric rendering.
- Perspective-correct decals for interactive objects.
- Dynamic sprite pipeline for monsters and items.

### Gameplay Mechanics
- Grid-based exploration.
- Timing-based combat with Attack and Defend/Parry.
- Inventory and leveling systems.

## Current Status

Transitioned to 1990s Modern state with full texture support. 
Priorities are:
1. Procedural map generation
2. Save state persistence (localStorage)
3. Advanced puzzle mechanics
